# Snake in Many Programming Languages

This repository contains implementations of the classic **Snake Game** in different programming languages and frameworks.

The project is part of a video series where the same game is built multiple times using different languages. The goal is to explore:

- language design differences
- libraries and frameworks
- code structure
- development experience

Each implementation aims to keep the gameplay similar while respecting the idioms of each language.

---

## Project Goal

This project demonstrates how the **same game can be implemented across different ecosystems**.

Topics explored include:

- game loop design
- input handling
- language-specific patterns

The intention is not to create the most optimized version, but to make the code **easy to read and educational**.
